Reusable React components.

## 组件规范 | Component Guidelines

### 组件结构 | Component Structure
每个组件应包含：
Each component should contain:
- `ComponentName.jsx` - 组件逻辑 | Component logic
- `ComponentName.module.css` - 组件样式 | Component styles

### 命名规范 | Naming Convention
- 组件文件：大驼峰命名 `MyComponent.jsx`
- 样式文件：大驼峰命名 + `.module.css`
- Component files: PascalCase `MyComponent.jsx`
- Style files: PascalCase + `.module.css`
